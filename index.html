Void.JS

The unopinionated JavaScript framework

Unopinonated

void was designed to be very unopinionated, allowing you to modify it in any way you see fit. Don;t like the way something is done? Change it!

Lightweight

void is very lightweight, and doesn't come with a lot of bloat. This makes it perfect for small projects or large ones.

Powered by Webpack

void is powered by webpack, which means you can use all of the features that webpack has to offer. This includes hot module reloading, tree shaking, and more.
