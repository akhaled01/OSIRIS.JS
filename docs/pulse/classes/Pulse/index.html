Pulse

@akhaled01/void • Docs

@akhaled01/void / pulse / Pulse

Class: Pulse<T>

The Pulse class is a reactive data-binding utility that synchronizes changes to its internal data model with the DOM. It supports both objects and arrays, enabling reactivity and rendering of data in the DOM.

Type Parameters​

• T extends object | any[]

Type of the data model which can be an object or an array.
